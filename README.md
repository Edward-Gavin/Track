# Track
Prediction of the time when container trucks arrive at the port of Yantian.

## 数据预处理

每辆车从当前时间到达港口的时间差，作为本次预测的y值，首先定位港口坐标，这个坐标在实际测量的过程中可能有误差，因此需要查找一个范围作为港口的坐标，然后在每辆货柜车的行车记录中查找这个范围内的坐标，并以此来计算在当前位置到达闸口的时间差。
### 盐田港闸口位置
根据我们获取到的资料显示，GPS数据在地图上显示是有偏移的，我们使用GPS经纬度和地图经纬度其中的一个坐标作为确定闸口坐标位置的标准。我们确定闸口的位置坐标的方法是：根据7月1号的货柜车到港的入闸时间的记录，确定当前货柜车在当前时间的地图经纬度，作为闸口的位置坐标，由于在闸口位置上会有一定的误差，因此我们在匹配位置信息的时候确定了一个误差值，进行范围匹配，货柜车的行车GPS记录中到达这个位置范围内就认为货柜车到港。

## 涉及的特征提取
1.	假设每辆货柜车每次行走的路线是固定的，因此每次货柜车到闸口的路径认为是一样的，需将有行车记录的货柜车的轨迹进行分类，经过对盐田港交通的分析可知，盐田港有两个闸口，而到达这两个闸口的路径大概有三条，分别为：盐排高速，惠深沿海高速，明珠大道。
2.	通过货柜车所登记的车辆经营范围对货柜车进行分类，将每个类型的货柜车所对应的车牌号进行标记整理。
3.	货柜车当前的行车速度，这里的速度在货柜车的GPS行驶记录中有记载，但是GPS的行驶速度和货柜车的形式速度是一样的，并且在GPS的记录中，也可以发现这样的规律，所以我们使用一个特征作为这两个字段的特征。
4.	时间维度，比如周几也作为一个特征，进行标记。但是由于给出的时间比较少，只给出了一周的时间的数据，我们这里也给出这个特征比较小的权重。
5.	考虑到深圳市对外地车也有限行的政策，将货柜车的外地与本地也作为一个特征，进行标记。
